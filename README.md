# Sistema Inteligente JB

Primeira versão local para análise estatística de resultados do Rio/PT-Rio e Look Goiás.

## Como usar
1. Abra `index.html` no navegador.
2. Escolha a aba Look Goiás ou Rio/PT-Rio.
3. Adicione resultados manualmente ou clique em “Carregar exemplo”.
4. O sistema gera ranking de grupos, dezenas, bichos atrasados e milhares sugeridas.

## Observação
O sistema faz análise estatística baseada em histórico. Não garante acerto e não prevê resultado com certeza.
